# ⚡ OpenClaw — Quick Start

You're 3 commands away from a running AI agent.

## 1. Configure your API key

```bash
openclaw config
```

You'll need an API key from one of these providers:
- **Anthropic** (Claude) — https://console.anthropic.com
- **OpenAI** (GPT) — https://platform.openai.com

## 2. Start the gateway

```bash
openclaw gateway start
```

This runs in the background and handles all communication.

## 3. Open the chat

```bash
openclaw
```

That's it. You're talking to your AI agent.
