# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] — 2026-03-02

### Added
- Initial release
- macOS and Linux installer (`install.sh`)
- Windows installer (`install.ps1`)
- System requirement checks (OS, Node.js, disk space, network)
- Automatic OpenClaw installation from official npm registry
- Workspace creation with quickstart guide
- Existing installation detection (skip if already installed)
- Update check for existing installations

### Notes
- OpenClaw is downloaded from its official source — nothing is bundled
- No telemetry or tracking
- No account required
